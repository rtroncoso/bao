# Texture Packing Documentation

**1. OPTIONAL:** In case you have power of two textures you will need to trim them first (for the next step):

```bash
./scripts/trim.sh '\(.*\).png' '\1.png'
```

> *Note: You can revert this process by using:*

```bash
./scripts/pot.sh '\(.*\).png' '\1.png'
```

**2.** Convert graphics to multiple of 32 sizes (makes sure graphics align to the same grid):

```bash
# @{client/assets/graphics}
mkdir aligned
./scripts/align32.sh '\(.*\).png' 'aligned/\1.png'
```

**5. [GameFroot Texture Packer](https://github.com/Gamefroot/Gamefroot-Texture-Packer)** commands:

**Install:** `npm install -g gamefroot-texture-packer`

Packing Static Graphics:

```bash
# @{client/assets/graphics/aligned}
gf-pack -w 1024 -h 1024 --powerOfTwo --trim false -f pixi.js -fuzz 0% -n tilesets -p tilesets/ $STATIC_GRAPHICS
```

Packing Animated Graphics (no need for grid alignment):

```bash
# @{client/assets/graphics}
gf-pack -w 2048 -h 2048 --powerOfTwo --trim false -f pixi.js -fuzz 0% -n animations -p animations/ $ANIMATED_GRAPHICS
```
